CSS 文本

text-transform

none 对文本不做任何改动，将使用源文档中的原有大小写。
uppercase和lowercase 将文本转换为全大写和全小写字符。
capitalize只对每个单词的首字母大写。

text-decoration

  • none
  • underline:下划线
  • overline:在文本的顶端画一个上划线
  • line-through:则在文本中间画一个贯穿线
  • blink: 会让文本闪烁

CSS 字体

font-style

  • normal - 文本正常显示
  • italic - 文本斜体显示
  • oblique - 文本倾斜显示
