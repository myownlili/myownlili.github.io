微信小程序-1

原生框架的目录结构

  • pages

    • index：存放首页的页面文件
      • index.js ：逻辑
      • index.json：配置项
      • index.wxml：便签嵌套
      • index.wxss：样式
    • logs：日志页面相关文件

  • utils：可要可不要，帮助文档相关JS

  • app.js：本小程序的入口文件和全局文件

  • app.json：全局配置文件，如整个项目的标题名称

  • app.wxss:全局样式文件

  • project.config.json：基础配置、环境配置

  • sitemap.json：配置小程序及其页面是否循序被微信索引

    小程序配置文件

    app.json

    “pages”

    本小程序有哪些页面
    一 一对应pages文件夹中的路径

    “window”

    定义小程序所有页面的顶部背景颜色、文字颜色定义等

    -backgroundTextstyle ：下拉刷新的所展现的文字颜色
    -navigationBarBackgroundColor ：导航栏背景颜色
    -navigationBarTitleText ：导航栏标题名称
    -navigationBarTextstyle ：导航栏文字颜色（black or white）
    -enablePullDownRefresh ：全局下拉刷新

      enablePullDownRefresh:true

-backgroundColor ：下拉刷新页面的背景颜色

“tabbar”

类似淘宝底部页面的切换
在app.josn中添加：输入tabBar回车自动生成如下代码

"tabBar": {
    "list": [{
      "pagePath": ",", //要跳转页面的页面路径
      "text": "text", //标题
      "iconPath": "iconPath", //未选中图标
      "selectedIconPath": "selectedIconPath" //已选中图标
    }]
  }

<app.josn>
如何改变此处标题颜色？在list的同级下编写

  "tabBar": {
    "list": [{
      ...
    }],
    "color":"#0094ff",
    "selectedColor": "#000000"//只能使用16进制
  }

<app.josn>
如何改变相对位置？

  "tabBar": {
    "list": [{...}],
    "color":"#0094ff",
    "selectedColor": "#000000",
    "position":"top" //未设置的话默认在bottom，tabbar在顶部时默认不显示图标
  }

<app.josn>

page.json用法与app.json中的Windo类似，其他使用参考小程序文档

数组和对象循环

列表循环

wx:for=“”
wx:for-item=”循环项的名称”
wx:for-index=”循环项的索引”

// pages/test/test.js
Page({

    data: {
        list:[
            {
                id:0,
                name:"小李"
            },
            {
                id:1,
                name:"amy"
            },
            {
                id:3,
                name:"tony"
            }
        ]
    }
})
<view>
    <view wx:for="{{list}}" 
    wx:for-item="item" 
    wx:for-index="index">
    索引：{{index}}
    --
    值：{{item.name}}
    </view>
</view>

wx:key=”唯一的值”提高列表渲染的性能
1 wx:key绑定个普通的字符串的时候 那么这个字符串名称肯定是循环数组中的对象的 唯一属性
2 wx:key ="*this" 就表示I你的数组是一个普通的数组 *this 表示是循环项
普通数组[1,2,3,44,5]

<view>
    <view wx:for="{{list}}" 
    wx:for-item="item" 
    wx:for-index="index"
    wx:key="id">
    索引：{{index}}
    --
    值：{{item.name}}
    </view>
</view>

3当出现数组的嵌套循环的时候尤其要注意 以下绑定的名称不要重名
wx:for- item=”item" wx : for- index=”index"
4默认情况下我们不写
Wx: for- item=" item" WX : for - index=”index"
小程序也会把循环项的名称和索引的名称item 和index
只有一层循环的话(wx :for item=" item" wx: for- index="index") 可以省略

对象循环

Wx :for="" wx:for-item="对 象的值" wX : for- index="对象的属性”
==循环对象的时候最好把item和index的名称都修改一下==
wx: for-item= “value”
WX : for- index= “key’

<view>
    <view>对象循环</view>
    <view wx:for="{{person}}" 
    wx:for-item="value" 
    wx:for-index="key"
    wx:key="age">
        属性：{{key}}
        --
        值：{{value}}
    </view>
</view>
