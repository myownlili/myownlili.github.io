上学的小李
  • ActivityTest:项目名
  • com.example.activitytest:包名（默认值）
  • 创建活动：com.example.activitytest->New->Activity->Empty Activity
  • Generate Layout File：自动创建一个对应的布局文件
  • Launcher Activity:自动将**Acitivity设置为当前项目的主活动
  • 代码切换：
  • setContentView():给当前活动加载布局，一般要传入id,如R.layout.first_layout
  • android:name指明具体注册的活动
  • 运行目标活动：在AndroidManifest.xm文件夹中，该活动的activity内添加intent-filter 标签，并在其中添加<action android:name="android.intent.action.MAIN"/> 和<category android: name="android.intent.category.LAUNCHER" />
  • android:label指明活动中标题栏的内容，注：给著活动指定label也会成为启动器中应用程序的名称
  • 点击事件+Toast:
  • findViewById()方法获取在布局文件中定义的元素，然后返回一个View对象
  • Toast的用法非常简单，通过静态方法makeText() 创建出一个Toast 对象，然后调用show()将Toast显示出来就可以了。
函数名 参数
makeText() 1.Context ，也就是Toast要求的上下文，由于活动本身就是一个Context 对象，因此这里直接传入FirstActivity.this 即可。
2.Toast显示的文本内容
3.Toast显示的时长，有两个内置常量可以选择Toast.LENGTH_SHORT 和Toast.LENGTH_LONG
  • 创建Menu资源文件：1.在res目录下创建menu文件夹(res->New->Directory->menu)
                 2.在menu文件夹创建main的菜单文件(menu->New-> Menu resource file)
